📖 Abstract

OCW-PW adalah platform open courseware untuk mata kuliah Pemrograman Web di Universitas IPWIJA, Jakarta. Platform ini dibangun menggunakan DonatJS — sebuah zero-dependency, no-build-step, JSON-driven micro-framework yang merender data terstruktur sebagai antarmuka web dinamis sepenuhnya di browser.

Courseware ini mencakup 16 modul pembelajaran terstruktur mulai dari HTML5 semantics hingga Single Page Application architecture, dilengkapi dengan integrated quiz engine dan certificate verifier.
